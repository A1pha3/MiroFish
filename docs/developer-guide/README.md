# 开发者指南

> 深入了解 MiroFish 的技术实现和开发方法

本指南面向：开发者、贡献者、二次开发者

## 学习目标

完成本指南学习后，你将能够：

### 基础目标（必掌握）

- [ ] 搭建本地开发环境
- [ ] 理解项目的基本架构
- [ ] 掌握开发工作流程
- [ ] 能够进行简单的功能开发

### 进阶目标（建议掌握）

- [ ] 理解核心服务实现原理
- [ ] 掌握前后端架构设计
- [ ] 能够进行中等复杂度功能开发
- [ ] 理解测试和部署流程

### 专家目标（挑战）

- [ ] 能够进行架构设计
- [ ] 能够贡献核心代码
- [ ] 能够指导其他开发者

## 常见问题

### Q: 后端启动失败？

**检查清单**：
1. Python 版本是否正确 (3.11-3.12)
2. 虚拟环境是否激活
3. 依赖是否安装完成 (`uv sync`)
4. `.env` 文件配置是否正确
5. 端口是否被占用 (`lsof -i :5001`)

### Q: 前端构建失败？

**检查清单**：
1. Node.js 版本是否 ≥18
2. `node_modules` 是否存在
3. 尝试清除缓存：`rm -rf node_modules && npm install`

### Q: API 调用跨域？

**开发环境**：已配置 CORS，应该正常工作

**生产环境**：需要配置 Nginx 反向代理

### Q: 模拟进程卡住？

**排查方法**：
1. 检查后端日志
2. 检查模拟进程状态 (`ps aux | grep python`)
3. 检查 IPC 目录是否存在
4. 尝试手动停止并重新启动
