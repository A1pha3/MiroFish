## 什么是 MiroFish？

### 核心定义

**MiroFish** 是一款基于**多智能体技术**的新一代 AI 预测引擎。它通过提取现实世界的种子信息（如突发新闻、政策草案、金融信号、文学作品），自动构建出高保真的**平行数字世界**。在此空间内，成千上万个具备独立人格、长期记忆与行为逻辑的智能体进行自由交互与社会演化。

### 核心价值主张

> **让未来在数字沙盘中预演，助决策在百战模拟后胜出**

| 价值维度 | 传统方式 | MiroFish 方式 |
|---------|---------|---------------|
| **成本** | 高（真实试错） | 低（数字沙盘） |
| **风险** | 高（不可逆） | 零（可重复） |
| **速度** | 慢（等待现实发生） | 快（模拟加速） |
| **深度** | 浅（表面观察） | 深（智能体互动） |
| **广度** | 有限（样本受限） | 广（大规模智能体） |

### 场景详解

#### 场景 1：舆情预测

**输入数据**：突发新闻报道、政策公告、社交媒体讨论

**预测输出**：
- 各利益相关方的反应模式
- 信息传播路径和关键节点
- 舆情演化方向和拐点识别
- 舆论干预效果评估

**实际应用**：企业危机公关预案、政策发布前评估

#### 场景 2：金融分析

**输入数据**：财报数据、市场新闻、行业分析、交易数据

**预测输出**：
- 市场情绪变化趋势
- 资产价格走势预测
- 系统性风险事件评估
- 投资组合压力测试

**实际应用**：投资决策辅助、风险对冲策略

#### 场景 3：创意仿真

**输入数据**：小说文本、历史事件、剧本设定

**预测输出**：
- 人物行为模式推演
- 多种可能结局探索
- 剧情分支分析
- 人物关系演化

**实际应用**：创作辅助、教育娱乐

## 技术特点

### 1. 知识图谱驱动 (GraphRAG)

**什么是 GraphRAG？**

GraphRAG (Graph Retrieval-Augmented Generation) 是一种结合知识图谱和检索增强生成的技术。相比传统 RAG，它能：

| 特性 | 传统 RAG | GraphRAG |
|------|---------|----------|
| 信息组织 | 独立文本块 | 实体关系网络 |
| 检索方式 | 向量相似度 | 多跳关系遍历 |
| 上下文理解 | 局部文本块 | 全局关联视角 |
| 时序支持 | 无 | 支持时序记忆 |

**MiroFish 中的应用**：

**命令类型定义**：

```python
class CommandType(str, Enum):
    INTERVIEW = "interview"           # 单个智能体采访
    BATCH_INTERVIEW = "batch_interview"  # 批量采访
    CLOSE_ENV = "close_env"           # 关闭环境

class IPCCommand:
    command_id: str      # 唯一标识符 (UUID)
    command_type: CommandType
    args: Dict[str, Any]  # 命令参数
    timestamp: str         # 时间戳
```
