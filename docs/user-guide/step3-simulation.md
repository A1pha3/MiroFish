## 核心概念：OASIS 模拟原理

### 什么是 OASIS？

**OASIS** (Open Agent Social Interaction Simulation) 是一个专业的社交媒体模拟引擎，它创造了一个**数字社交沙盘**：

```
┌─────────────────────────────────────────────────────────────┐
│                    OASIS 核心特性                            │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 真实平台模拟                                            │
│     ┌─────────────────┐      ┌─────────────────┐           │
│     │  Twitter        │      │  Reddit         │           │
│     │  · 推文/转发    │      │  · 帖子/评论    │           │
│     │  · 点赞/关注    │      │  · 投票/回复    │           │
│     │  · 280 字限制   │      │  · 长讨论       │           │
│     └─────────────────┘      └─────────────────┘           │
│                                                              │
│  2. 独立智能体                                              │
│     每个智能体有:                                           │
│     · 独立人格 (Profile 定义)                               │
│     · 个人记忆 (Zep 图谱存储)                               │
│     · 社交关系 (关注网络)                                   │
│     · 行为模式 (LLM 驱动)                                   │
│                                                              │
│  3. 动态演化                                                │
│     · 关系会随互动变化                                      │
│     · 观点会受他人影响                                      │
│     · 群体结构自然形成                                      │
│                                                              │
│  4. 可观测性                                                │
│     · 所有动作可追踪                                        │
│     · 随时可以访谈智能体                                    │
│     · 关系网络可视化                                        │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 智能体动作类型

#### Twitter 动作

| 动作 | 说明 | 消耗 | 影响 |
|------|------|------|------|
| **CREATE_POST** | 发布新推文 | 高 | 开启新话题 |
| **LIKE_POST** | 点赞推文 | 低 | 增加亲密度 |
| **RETWEET** | 转发推文 | 低 | 扩散信息 |
| **QUOTE_POST** | 引用转发 | 中 | 加入自己的观点 |
| **FOLLOW** | 关注用户 | 低 | 建立关注关系 |
| **DO_NOTHING** | 跳过本轮 | 无 | 休息 |

#### Reddit 动作

| 动作 | 说明 | 消耗 | 影响 |
|------|------|------|------|
| **CREATE_POST** | 发布帖子 | 高 | 开启新讨论 |
| **CREATE_COMMENT** | 发表评论 | 中 | 参与讨论 |
| **UPVOTE** | 投票支持 | 低 | 增加帖子可见性 |
| **DOWNVOTE** | 投票反对 | 低 | 表达不认同 |

### 状态说明

| 状态 | 说明 | 可执行操作 |
|------|------|-----------|
| **CREATED** | 模拟已创建，等待准备 | 开始准备 |
| **PREPARING** | 正在准备环境和 Profile | 等待完成 |
| **READY** | 准备就绪，可以启动 | 开始模拟 |
| **RUNNING** | 正在运行中 | 暂停、停止、访谈 |
| **PAUSED** | 已暂停 | 继续、停止 |
| **COMPLETED** | 自然完成所有轮次 | 查看结果、访谈 |
| **STOPPED** | 手动停止 | 继续剩余轮次 |
| **FAILED** | 运行失败 | 查看错误日志 |

## 故障排查

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| **模拟很慢** | 智能体多、LLM 慢 | 减少智能体、使用更快模型 |
| **模拟中断** | 后端服务异常 | 检查服务状态，查看日志 |
| **动作异常** | LLM 生成问题 | 检查 API 配置 |
| **无法访谈** | IPC 通信问题 | 检查模拟进程状态 |

### 问题诊断决策树

```
遇到问题?
    │
    ▼
模拟无法启动?
    │ YES
    ├─→ 检查后端服务状态
    ├─→ 检查配置文件
    └─→ 查看错误日志
    │
    NO
    ▼
模拟速度很慢?
    │ YES
    ├─→ 智能体太多？减少数量
    ├─→ LLM 太慢？切换模型
    └─→ 网络延迟？检查连接
    │
    NO
    ▼
智能体行为异常?
    │ YES
    ├─→ Profile 问题？重新生成
    ├─→ LLM 问题？调整温度参数
    └─→ 查看具体动作分析原因
    │
    NO
    ▼
访谈无响应?
    │ YES
    ├─→ 检查 IPC 进程
    ├─→ 查看智能体状态
    └─→ 重启模拟
```
